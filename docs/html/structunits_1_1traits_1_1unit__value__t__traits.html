template<typename T>
struct units::traits::unit_value_t_traits< T >

Trait for accessing the publically defined types of units::unit_value_t_traits

The units library determines certain properties of the unit_value_t types passed to them and what they represent by using the members of the corresponding unit_value_t_traits instantiation.
